## What is this?
A banking database system built with real cybersecurity features — not just basic CRUD. It detects intrusions, logs every action, and alerts on suspicious activity.

---

## 🛡️ Security Features
| Feature | What it does |
|---|---|
| Intrusion Detection | Flags failed logins & unusual transactions |
| Audit Logging | Every action is timestamped & stored |
| Role-Based Access | Admin vs User separation |
| Threat Alerts | Auto-alerts for suspicious behavior |

---

## 🗄️ Database Tables
`Users` · `Accounts` · `Transactions` · `Audit_Log` · `Alerts`

- Brute force login attempts
- Sudden large transactions
- Unauthorized access attempts
- Any data modification (audit trail)
